# footbal-game (footy-draft)

Настольно-цифровой драфт футболистов: настройка режима, жеребьёвка, драфт тренеров и игроков, подсказки, итоги и текст для симуляции в чате с моделью.

- **Стек:** React 19, TypeScript, Vite 8.
- **Версия приложения:** задаётся в `src/shared/config/version.ts` (`APP_VERSION`) и в `package.json`; сохранения в `localStorage` привязаны к `APP_VERSION`.

## Версия 18 — список изменений

### Подсказка «Лучший состав»

- У каждого игрока в списке (старт и запас, клубы и сборные) добавлена кнопка **«Копировать»**: в буфер попадает русское имя и фамилия для ввода в драфт (если русского нет — латиница из данных).
- Краткая индикация **«Скопировано»** после успешного копирования; запасной ввод в буфер через `textarea`, если `clipboard` недоступен.

### Экран загрузки (жеребьёвка)

- Под прогрессом показываются **флаги** всех источников раундов текущей партии (сборные / клубы / хаос с учётом типа раунда).
- При **режиме разработки** (чекбокс при старте партии) длительность заставки **5 секунд** вместо 10.

### Драфт игроков (во время партии)

- Доски команд в **карусели**: в окне видно **две** доски одновременно (ширина каждой ~половина области); при смене хода прокрутка ставит **команду с текущим ходом** у левого края, рядом видна **следующая** в очереди.
- Кнопки **‹ ›** и **горизонтальный свайп** — просмотр остальных команд.

### Выбор схемы поля (после драфта тренеров)

- Как прежде: **сетка карточек** со всех доступных схем (`auto-fill`, мини-превью поля).

### Капитан команды

- После полного заполнения составов всех команд — **на том же экране драфта** (доски команд и карусель) фаза **«капитан»**: у активной команды нажмите **карточку игрока на поле** — он станет капитаном (как выбор слота при драфте).
- Для **нейро-команд** капитан назначается **автоматически** в приоритете:
  1. **5★ вратарь** (слот GK с пятью звёздами);
  2. затем **любой игрок с 5★** (по порядку отображения схемы);
  3. если пятизвёздочных нет — **то же для 4★** (сначала вратарь, затем полевые);
  4. далее **3★, 2★, 1★** по тому же правилу;
  5. если у игроков нет рейтинга в данных — **первый занятый слот** по порядку схемы.
- На **странице итогов** у капитана отображается повязка; в блоке про симуляцию в чате добавлено напоминание: **«Учти влияние капитана команды»**; в текст **«Скопировать всё»** включены строка про капитана по командам и пометка `[капитан]` у позиции в списке.
- Итоговый экран сделан **компактнее** (отступы, шрифты, кнопки); **без** золотых/серебряных визуальных акцентов по звёздам у игроков и тренера в списке.

### Прочее

- Старые сохранения без поля капитана при загрузке в фазе «игра завершена» дополняются капитаном по той же логике приоритета звёзд, чтобы экспорт и подсказки были полными.
